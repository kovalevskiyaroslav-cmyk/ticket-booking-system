# Ticket Booking System

## Project Description

A REST-based ticket booking system for managing events and seat reservations.

## Core Functionality

- User management
- Event management
- Venue management
- Seat management
- Ticket booking
- Order processing
- Payment processing (asynchronous)
- Event filtering (by city, price, available seats)
- Transactional order creation
- Concurrency handling (race condition prevention)
- Bulk seat and event creation
- In-memory caching
- Load testing support
- Docker-based deployment
